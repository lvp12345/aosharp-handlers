# AOSharp Handlers & Managers

A collection of AOSharp combat handlers and managers for Anarchy Online.

## Key Features

- **Removed PvP checks** - Handlers work without PvP restrictions
- **Instant Agent FP casting on zone** - Agent handler attempts to cast set FP nano immediately upon zoning, with continuous retry for up to 3 seconds until successful

## Handlers Included

- Agent Combat Handler
- MP Combat Handler
- NT Combat Handler
- Engineer Combat Handler
- And more...

## Managers Included

- Loot Manager
- Assist Manager
- Sync Manager
- And more...
